CSV Columns

Each row in the exported file represents one crash and includes the following columns:

  • Crash ID, Case Number, Date, Time, Day of Week
  • County FIPS, County, City/Place, Route ID, Route Type Code
  • Severity Code, Crash Type Code, Manner of Collision Code
  • Light Condition Code, First Harmful Event Code, Location of First Harmful Event
  • Junction Related, School Bus Related, Work Zone Related
  • Alcohol Involvement, Drug Involvement
  • Motor Vehicles, Motorists, Non-Motorists, Non-Fatal Injured, Fatalities
  • Source of Info Code, Latitude, Longitude, Created, Modified

Coded values (e.g., Severity Code) use MMUCC numeric codes. Refer to the MMUCC 5th Edition specification for code definitions.
